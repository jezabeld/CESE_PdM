Inicializa el giroscopio y acelerometro en el modo de alimentación seleccionado.

Realiza la inicialización del dispositivo en el modo de alimentación seleccionado, testea la correcta comunicación con el dispositivo y realiza una calibración inicial del mismo.

Parameters
[in,out]gyroEstructura del dispostivo a inicializar.
[in]hi2cHandler de I2C para la conexión con el dispositivo.
devAddressDirección del dispositivo para la conexión por I2C.
modeModo de alimentación y funcionamiento seleccionado.
Returns
Estado de la inicialización: GYRO_OK si la incialización finalizó con éxito, GYRO_ERROR si hubo problemas de conexión o comunicación con el dispositivo o el modo de alimentación seleccionado es incorrecto o no definido. GYRO_ERROR También puede indicar errores de comunicación con el dispositvo debido a una dirección errónea del mismo o que el dispositivo conectado no es el modelo MPU-6500. Otros modelos requieren configuraciones diferentes y no se recomienda el uso de este driver.
