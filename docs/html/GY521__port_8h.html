Lee un registro (o un conjunto de registros contiguos) del dispositivo conectado por I2C en la dirección seleccionada.

Parameters
[in]hi2cHandler de I2C para la conexión con el dispositivo.
devAddressDirección del dispositivo para la conexión por I2C (en formato 7 bits).
[out]dataPuntero donde guardar la información leída del registro.
regAddressDirección del registro a leer (en formato 7 bits).
sizeReadTamaño a leer del registro seleccionado en bytes. Valores válidos entre 1 y 14 bytes.
Returns
false si la comunicación I2C fue exitosa, true en caso contrario.
